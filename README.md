# Personal ChordPro Viewer

A lightweight personal ChordPro viewer, editor, and setlist manager for live performance use.

## Demo

https://over-keys.github.io/Personal-ChordPro-Viewer/

## Features

- Add, edit, import, and export songs in ChordPro format
- Search songs by title, artist, and tags
- View clean performance-ready chord charts
- Transpose songs and choose automatic, sharp, or flat chord notation
- Save song-level Key, Sign, and Auto Scroll settings from Song Controls
- Auto Scroll with adjustable duration and chart-based `[S]tart` / `[E]nd` markers
- Setlists with ordered song references, duplicate songs allowed, and Prev / Next navigation
- Built-in YouTube links and search support
- Sticky notes on charts, with per-browser note storage and note color settings
- Chord formatting, colors, and font settings for readability during performance
- JSON backup and restore for songs, setlists, and sticky notes
