Personal ChordPro Viewer の使い方

Personal ChordPro Viewer は、ChordPro 形式の譜面を見やすく管理し、演奏時にも使いやすくするための個人用アプリです。

最初の準備 最初の1曲を入れる トップページ Edit画面 閲覧画面 自動スクロール 付箋 バックアップ 他のバージョンと譜面共有 著作権と利用上の注意 迷ったときの最短ルール

1. 最初の準備

https://over-keys.github.io/Personal-ChordPro-Viewer/ をブラウザで開いてください。ブックマークしておくと便利です。このサイトで作成した譜面はすべて、自分が今使っているブラウザ内にのみ保存されます。

途中で使うブラウザを変えると、保存済みの曲や付箋が見えなくなります。Export / Import 機能はありますが、できるだけ同じパソコンと同じブラウザを使ってください。

2. 最初の1曲を入れる

初めてのときは、次の順番がいちばん簡単です。

  1. トップページ右上の Add Song を押します。
  2. Slug を入れます。短く、重複しにくい文字列にしてください。使えるのは英数字とハイフンのみです。例: hot-chili-love-song
  3. ChordPro 欄に譜面を入力します。
  4. Title と Artist を確認します。
  5. 必要なら Tags と YouTube links を入れます。
  6. Save を押します。
いちばんのコツは、Title や Artist を先に手入力するより、まず ChordPro を入力することです。
ChordPro に {title: 曲名} と {subtitle: アーティスト名} が入っていれば、Title と Artist は自動で反映されます。Slug が自動で候補入力されるのは、Title が英語・ローマ字など半角英数字ベースのときだけです。日本語タイトルでは自動では入らないので、Slug は手動で入力してください。

入力例

3. トップページの使い方

トップページでは、曲の一覧を見たり、新しい曲を追加したり、全曲の Export / Import を行えます。検索欄では title / artist / tags をまとめて検索できます。入力後に Enter を押すと検索され、元に戻したいときは Clear を押します。曲カード本体を押すと閲覧画面に入り、Edit を押すと編集画面に入ります。

Export all / Import all

  • Export all: 曲ライブラリ全体を JSON に書き出します。
  • Import all: JSON を読み込みます。同じ slug の曲は上書きされ、JSON に含まれていない既存曲はそのまま残ります。

4. Edit画面の使い方

Edit画面では、曲の追加・修正・1曲単位の Export / Import を行います。上部には Cancel / Import / Export / Save があり、保存済みの曲では Delete も使えます。

よく使うボタン

  • Save: 入力内容を保存します。
  • Delete: その曲を削除します。
  • Export: 1曲分の JSON を書き出します。
  • Import: 1曲分の JSON を読み込んで、編集フォームに載せます。
Import しただけでは、まだ保存は確定しません。
内容を確認して、必要なら直してから Save を押してください。別の曲をひな形にして新しい曲を作るときにも便利です。

YouTube links

YouTube は 1行に1件 入れるのが分かりやすいです。通常の URL、youtu.be 形式、動画 ID が使えます。開始位置付きリンクも使えます。たとえば https://www.youtube.com/watch?v=XXXXXXXXXXX&t=90 と書くと 90 秒から再生され、https://youtu.be/XXXXXXXXXXX?t=90 の形でも使えます。

5. 閲覧画面の使い方

曲を開くと閲覧画面になります。中央に譜面、右側に設定パネルが出ます。右側では、フォントやコード表記、YouTube、付箋、自動スクロールをまとめて操作できます。

基本操作

  • Back: トップページへ戻ります。
  • Edit: その曲の編集画面へ移動します。

表示調整

閲覧画面の右側パネルでは、上から順に次の調整ができます。

  • Key: 譜面全体のキーを上下に移調します。
  • Sign: シャープ系、フラット系など、音名の表記を切り替えます。
  • Auto Scroll: 自動スクロールの開始・停止、時間設定、保存を行います。
  • YouTube: 登録した動画を開いて再生できます。このGitHub Pages版では、動画は右下にフロート表示で再生されます。ローカル版では、動画は別ウィンドウで開かれます。登録されていない場合は、Title と Artist でYouTube検索することもできます。
  • Notes: 付箋の書き出し・読み込みができます。
  • Chord Formatting: テンション、スラッシュコード、add・sus など、コード表記の見た目を調整できます。
  • Font Settings: 歌詞とコードのサイズ、フォントを調整できます。

6. 自動スクロールの使い方

右側の Auto Scroll には、Start / Pause、Reset、Save、分・秒入力、3:00 / 4:00 / 5:00 プリセットがあります。

まずはこの順番がおすすめ

  1. 譜面左側の S / E マーカーを動かして、スクロールする範囲を決めます。
  2. まずは 3:00 / 4:00 / 5:00 のどれか近いものを押します。
  3. Start を押します。
  4. 演奏しながら必要に応じて調整します。
  5. 演奏が最後まで終わったら、分・秒入力欄に戻って、実測値を確認します。
  6. 実測値が適切な長さであることが確認できたら（あるいは手動で調整後）Save を押します。

開始・停止のしかた

  • 右側の Start / Pause ボタン
  • Space キー
  • 譜面の左クリック

このどれでも開始・停止を切り替えられます。ただし、付箋を操作しているときや S / E マーカーをつかんでいるときは、左クリックはスクロール切替には使われません。

実用的な使い方
最初から完璧な時間を入れようとしなくて大丈夫です。まず近い時間で始めて、演奏しながら少しずつ合わせるのがいちばん楽です。

自動補正

自動スクロール中に手でスクロールすると、その方向に応じて速度が補正されます。

  • 上へ戻す → 少し遅くしたい
  • 下へ送る → 少し速くしたい
  • 一時停止する → 少し遅めにしたい

つまり、演奏しながら「少し戻す」「少し送る」「必要に応じて一時停止する」を使うことで、だんだん合いやすくなります。

自動測定

スクロール範囲のかなり後ろまで進むと、アプリが実際にかかった時間を測定し、分・秒欄に候補として表示します（青くハイライトされます）。ただし、測定された時間は、その時点ではまだ曲の設定として保存されていません。その時間を今後も使いたい場合は、内容を確認したうえで Save を押してください。

7. 付箋の使い方

付箋は、譜面の上に直接メモを置ける機能です。歌い出しの注意、転調ポイント、カポ、リズムのメモなどに向いています。

付箋を置く

譜面の上で右クリックすると、その場所に新しい付箋を作れます。作成したら、そのまま入力してください。

付箋を閉じる・最小化する

  • ✓: 編集を終えて閉じる
  • _: 最小化する
  • ✎: 編集する
  • ×: 削除する

付箋を動かす・大きさを変える

  • ドラッグで移動できます。
  • 右下のハンドルでサイズ変更できます。
  • A− / A+ で文字サイズを変えられます。

付箋のバックアップ

右側の Notes 欄には、Export song、Export all、Import があります。

  • Export song: 今開いている曲の付箋だけ書き出します。
  • Export all: 全曲分の付箋を書き出します。
  • Import: 付箋 JSON を読み込みます。1曲分の JSON にも、全曲分の JSON にも対応しています。
Notes の Import は、全曲分 JSON を選べば実質的に付箋の Import all として動作します。
そのとき、JSON 内に含まれる slug のうち、今の曲ライブラリに存在しないものは、インポート前に一覧表示され、実際の取り込みでは自動的にスキップされます。存在する slug の付箋だけが読み込まれます。
曲データの Export / Import と、付箋の Export / Import は別です。完全なバックアップを取りたいときは、両方行ってください。

8. バックアップと移行

普段のおすすめ

曲を追加・修正した日は、最後に次の2つをやっておくと安心です。

  1. トップページで Export all
  2. どれかの曲を開いて、Notes の Export all

これで、曲データと付箋データの両方をバックアップできます。

別のPCに移したいとき

  1. 新しいPCでも https://over-keys.github.io/Personal-ChordPro-Viewer/ をブラウザで開く
  2. トップページで Import all をして曲データを戻す
  3. どれかの曲を開いて、Notes の Import で付箋を戻す
付箋の全曲分 JSON を読み込むときは、先に曲データを戻しておくのが安全です。まだ存在しない slug の付箋は、一覧表示されたうえでスキップされます。

9. 他のバージョンと譜面共有

この静的HTML版に加えて、自分のパソコンで動かすローカル版（動作が軽快・個人用）と、Googleスプレッドシート版（表示や保存は遅め・譜面共有用）も用意しています。必要に応じて自由に活用してください。

ネットワーク不調などの影響を受けにくく、最も安全なのはローカル版です。ふだん自分だけで使う場合は、ローカル版をおすすめします。ただし、ローカル版では YouTube の再生は別ウィンドウで行われます。

一方、譜面共有が必要な場合は、Googleスプレッドシート版を使ってください。Googleスプレッドシート版は、譜面の表示や保存に時間がかかりますが、共通リンクによって譜面を共有できます。付箋は個人ごとに管理されます。

10. 著作権と利用上の注意

本ツールは、利用者が適法に利用できるデータのみを対象としています。外部サービスの利用規約や著作権法に反するデータの転記・保存・共有・公開には使用しないでください。

11. 迷ったときの最短ルール

  • 新しい曲を入れる → Add Song
  • Title / Artist を楽に入れたい → 先に ChordPro を入力する
  • 自動スクロールが合わない → まず範囲、次に時間、最後に演奏しながら微調整
  • 付箋を置きたい → 譜面上で右クリック
  • 1曲だけ持ち出す / 取り込む → Edit画面の Export / Import と、Notes の Export song / Import
  • 全部バックアップする → トップページの Export all と、Notes の Export all
